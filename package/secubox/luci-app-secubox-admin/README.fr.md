# LuCI SecuBox Admin Control Center

Tableau de bord d'administration unifie pour les plugins de l'app store SecuBox avec surveillance et gestion du systeme.

## Installation

```bash
opkg install luci-app-secubox-admin
```

## Acces

LuCI > SecuBox > Admin Control

## Onglets

- **Panneau de controle** -- Tableau de bord d'administration central
- **Console Cyber** -- Console systeme orientee securite
- **Gestionnaire d'apps** -- Installer, mettre a jour et gerer les applications SecuBox
- **Mises a jour** -- Mises a jour de paquets et systeme disponibles
- **Profils** -- Gestion des profils utilisateurs et appareils
- **Skills** -- Configuration des skills d'automatisation
- **Sources de catalogue** -- Gestion des flux de catalogue d'apps
- **Feedback** -- Soumission de retours utilisateurs
- **Parametres d'apps** -- Parametres par application
- **Sante systeme** -- Surveillance de la sante du materiel et des services
- **Logs systeme** -- Visualiseur de logs centralise
- **Parametres avances** -- Configuration systeme de bas niveau

## Dependances

Package JavaScript/JSON pur -- aucune dependance d'execution supplementaire au-dela de LuCI.
