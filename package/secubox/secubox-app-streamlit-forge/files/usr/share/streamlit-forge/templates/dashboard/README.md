# Dashboard Template

Multi-page dashboard with metrics, charts, and data tables.

## Features
- 4-page navigation (Overview, Analytics, Data, Settings)
- Metric cards with delta indicators
- Line and area charts
- Data table with export functionality
- Configurable settings

## Pages
1. **Overview** - Key metrics and trend charts
2. **Analytics** - Detailed analysis with tabs
3. **Data** - Raw data explorer with CSV export
4. **Settings** - Customization options

## Usage
```bash
streamlitctl create mydashboard --from-template dashboard
streamlitctl start mydashboard
```
