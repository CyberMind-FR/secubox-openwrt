# Data Viewer Template

CSV/Excel data explorer with filtering and visualization.

## Features
- Upload CSV and Excel files
- Dynamic column filtering
- Numeric range filters
- Interactive data table
- Quick visualizations (histogram, scatter)
- Statistical summary
- Correlation matrix
- Filtered data export

## Tabs
1. **Table** - Filterable data grid with export
2. **Charts** - Histograms and scatter plots
3. **Statistics** - Summary stats and correlations

## Usage
```bash
streamlitctl create myviewer --from-template data-viewer
streamlitctl start myviewer
```
