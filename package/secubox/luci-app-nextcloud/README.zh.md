# LuCI Nextcloud

用于管理自托管 Nextcloud 实例的 LuCI Web 界面 -- 文件同步、日历和联系人。

## 安装

```bash
opkg install luci-app-nextcloud
```

## 访问

LuCI > 服务 > Nextcloud

## 选项卡

- **概览** -- Nextcloud 服务状态和实例信息
- **设置** -- 实例配置和管理

## RPCD 方法

服务：`luci.nextcloud`

## 依赖项

- `luci-base`
